# Tridha Shopping Center: Sales & KPI Analysis

## 🎯 Project Objective
The primary goal of this project was to analyze large-scale retail data for Tridha Shopping Center. By using advanced Excel techniques, I transformed raw sales data into an interactive dashboard to help the management track revenue, profit margins, and customer trends.

## 📊 Business Key Performance Indicators (KPIs)
In this dashboard, I focused on:
- **Total Revenue & Net Profit:** Overall financial health of the center.
- **Sales by Category:** Identifying top-performing product lines.
- **Monthly Sales Trend:** Spotting seasonal growth and dips.
- **Payment Method Analysis:** Understanding customer payment preferences (Cash/UPI/Card).

## 🛠️ Skills & Tools Demonstrated
- **Data Cleaning:** Used Power Query to remove duplicates and fix formatting errors in 15k rows.
- **Advanced Formulas:** Used `VLOOKUP`, `SUMIFS`, and `IF` logic for dynamic calculations.
- **Pivot Tables & Slicers:** Created interactive filters for Region, Date, and Category.
- **Data Visualization:** Designed professional Bar charts, Doughnut charts, and KPI cards.

## 📁 Repository Contents
- **Tridha_Sales_Data.xlsx**: The main Excel file with raw data, pivot tables, and the dashboard.

## 🚀 How to Explore
1. Download the `.xlsx` file from this repository.
2. Open it in Microsoft Excel.
3. Use the **Slicers** on the dashboard to interact with the data and see real-time updates.
